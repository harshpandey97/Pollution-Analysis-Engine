> AI-powered air pollution analysis dashboard with real-time insights and recommendations.

## 🚀 Features
- 📊 Interactive pollution data visualization
- 🤖 AI-generated insights using LLaMA 3.3
- 🏙️ Multi-city pollution comparison
- 📈 AQI trend analysis with Pandas
- 🌐 Web dashboard using Streamlit

## 🛠️ Tech Stack
| Tool | Purpose |
|------|---------|
| Python | Core language |
| Pandas | Data analysis |
| Matplotlib | Graphs & charts |
| Streamlit | Web dashboard |
| Groq AI | AI insights |

## 📁 Project Structure
pollution-analysis-engine/

├── main.py           # Data analysis + graphs

├── dashboard.py      # Streamlit web app

├── ai_insights.py    # Groq AI recommendations

├── data/

│   └── pollution_data.csv

└── requirements.txt

## 🔧 Setup
```bash
git clone https://github.com/HARSHPANDEY9756/pollution-analysis-engine
cd pollution-analysis-engine
pip install -r requirements.txt
python main.py
```
